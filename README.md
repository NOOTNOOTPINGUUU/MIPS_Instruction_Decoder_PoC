MIPS_Instruction_Decoder_PoC

A MIPS32 Instruction Decoder implemented in Python while self-studying computer architecture fundamentals.

Current Status:
- R-type instructions supported
- I-type (addi/lw/sw) instructions supported
- ALU implemented
- Control Unit implemented
- Work in progress

Supported Instructions

- add
- sub
- and
- or
- nor
- slt
- addi
- lw
- sw

Example
```assembly
$ add $t0 $t1 $t2
```
